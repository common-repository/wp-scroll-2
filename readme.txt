== Description ==

Scroll to top plugin is an simple and nice plugin with the standard scroll settings. You can use it on your website different sides and set different scrolling times.
It's very simple and functional at the same time. 

You can check more details about the plugin on this page

*   [WordPress scroll plugin](https://wpdevart.com/wordpress-scroll-plugin-how-add-back-to-top-button/)

### Features of WordPress Back to top plugin

*   **Easy install and use**
*   **Enable or Disable back to top**
*   **Upload image or set url**
*   **Scrolling time**
*   **Position of the button**
*   **Padding element**
*   **User friendly back-end**
*   **Tested on popular WordPress themes**
